# Living Expense Predictor

A simple Python-based application that estimates monthly living expenses for students, employees, or anyone planning to move to a new city or hostel.

The project predicts approximate expenses based on accommodation type, food choices, transportation, utilities, and other monthly costs.

---

## Project Overview

Moving to a new city requires proper budgeting. This project helps users estimate their expected monthly and yearly expenses before relocating.

The application uses a rule-based approach to calculate expenses according to user preferences.

---

## Features

- City-based expense estimation
- Hostel, apartment, and house cost calculation
- Single and shared room options
- Food expense prediction
- Transportation cost estimation
- Monthly and yearly expense calculation
- Expense breakdown report
- Budget suggestion based on total cost

---

## Technologies Used

- Python
- Conditional Statements
- Variables
- User Input Handling
- Basic Mathematical Operations
